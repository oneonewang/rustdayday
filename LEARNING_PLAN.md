# Rust 系统学习计划

> 完整的分阶段学习规划，详见 [Rust 学习项目总览](./README.md)。

---

## 🦀 6 个阶段总览

| 阶段 | 主题 | 周期 | 核心目标 |
|------|------|------|----------|
| Stage 1 | 基础语法与工具链 | 1–2 周 | 跑通 cargo，理解所有权 / 借用 |
| Stage 2 | 结构化代码与错误处理 | 1–2 周 | struct / enum / 模式匹配 / `Result` |
| Stage 3 | 泛型、Trait、生命周期 | 2–3 周 | 写出"像 Rustacean"的抽象 |
| Stage 4 | 智能指针 / 闭包 / 迭代器 | 1–2 周 | 函数式风格与常用模式 |
| Stage 5 | 并发与异步 | 2 周 | Tokio、async/await |
| Stage 6 | 实战项目 | 4+ 周 | CLI / Web / WASM 端到端 |

每阶段详细规划见 [README.md](./README.md) 与各阶段目录。
